图床
==================
图床创建过程
------------------------------------
使用`jsDeliver+github`搭建，`github`存储图片,`jsDeliver`做`cdn`加速访问。
github的`asset`为图片目录，通过markdown通过cdn访问的语法：
```https://cdn.jsdelivr.net/gh/你的用户名/你的仓库名@发布的版本号/文件路径```
